# 对象划分
## 1.葫芦娃CalabashBrother
        
使用枚举类型，将葫芦娃限定为不同的7个，并在enum中定义了一系列葫芦娃自身可完成的事
包括与别人的比较，报名，喊出移动方向等

## 2.排序Sort   

Sort中只有一个空的sort()方法，而冒泡排序和二分排序则以继承和覆盖的方式分别重写了sort()方法
这使得排序人只需使用在选择好Sort子类之后直接使用Sort类的sort()方法就可以执行不同的排序算法

## 3.爷爷Grandpa

爷爷是葫芦娃的指挥者，他可以创造出7个葫芦娃，也可以命令创造出的葫芦娃随机站队，或是采取不同的排序算法
对葫芦娃进行排序，而在排序时爷爷进行葫芦娃之间的比较则依赖于葫芦娃自身同别人比较的结果
